## Portfolio (Next.js)

This project rebuilds the existing portfolio in Next.js with a cleaner structure, while keeping the same look and feel.

## Getting started

- Install dependencies: `npm install`
- Run locally: `npm run dev` (defaults to http://localhost:3006)
- Build: `npm run build`
- Start production server: `npm run start`

## Notes

- Images are served from `public/images`.

## Deploy to Vercel

1. Push this repository to GitHub.
2. In Vercel, create a new project and import the repo.
3. Framework preset: Next.js (auto-detected).
4. Build command: `npm run build`.
5. Output: default (Next.js).
6. Deploy.
